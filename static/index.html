Bunchin API Proxy

Este serviço atua como um proxy seguro para a API Java do Bunchin, facilitando o acesso e roteamento de requisições HTTP para os seguintes recursos:

  • Funcionário: GET, POST, PUT, DELETE, busca por nome ou CPF
  • Login: POST para autenticação
  • Ponto: Gerenciamento de registros de ponto
  • Reset de Senha: Solicitação e redefinição de senha

Todas as requisições para /api/... são encaminhadas automaticamente para a API Java, com autenticação básica e preservação dos métodos e parâmetros originais.
